#四无浪子Blog
 
这是一个利用 Jekyll-Bootstrap 创建的Blog。 主要是为了学习，用Markdown来写文档。用Markdown写好的文档，再用pandoc可以转换为pdf，
HTMl，doc等格式。这样就可以方便的发布了。要修改的时候也比较简单。可以像代码一样，通过文本比较修改。

http://johnmacfarlane.net/pandoc/README.html
